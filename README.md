# cornBox

CronBox — учебный планировщик выполнения shell-команд по расписанию, написанный на Go.

Проект предназначен для изучения архитектуры небольших Go-приложений, работы с пакетами, парсинга конфигурации, конкурентности и взаимодействия компонентов системы.

## Возможности

* чтение конфигурации из JSON-файла;
* хранение списка задач в памяти;
* запуск shell-команд по cron-расписанию;
* логирование результатов выполнения;

```
## Формат конфигурации

Пример файла `config.json`, который должен находиться в корне проекта:

```json
{
  "jobs": [
    {
      "schedule": "*/5 * * * *",
      "command": "echo Hello, CronBox!"
    }
  ]
}
```

## Сборка

Переместиться в корень проекта:
```bash
cd cronBox
```

Собрать проект:

```bash
go build -o bin/cronBox ./cmd/cronBox/
```

После успешной сборки будет создан исполняемый файл `cronBox`.

## Запуск
```bash
./bin/cronBox
```
